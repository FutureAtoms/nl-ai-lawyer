# Search Dutch Case Law (Rechtspraak)

Search the Dutch case law database (rechtspraak.nl) for relevant jurisprudence.

## Skill and Tools

Use skill: **dutch-case-law-research**

Use MCP tools:
- `caselaw_search_rechtspraak` -- primary search tool for Dutch case law
- `caselaw_fetch_uitspraak` -- to retrieve full text of a specific ruling
- `legislation_search` -- to find the statutory provisions referenced in case law

## Workflow

### Step 1: Query Preparation

- If the query is in English, translate key legal terms to Dutch for better search results
- Identify relevant legal concepts and synonyms
- Common translations to consider:
  - Contract breach = wanprestatie / tekortkoming in de nakoming
  - Tort / negligence = onrechtmatige daad
  - Damages = schadevergoeding
  - Employment dismissal = ontslag / opzegging
  - Lease = huur
  - Force majeure = overmacht
  - Good faith = redelijkheid en billijkheid
  - Unjust enrichment = ongerechtvaardigde verrijking

### Step 2: Execute Search

Use `caselaw_search_rechtspraak` with:
- The search query (use Dutch terms for best results)
- Court filter if the `court` argument is provided:
  - `Hoge Raad` -- Supreme Court (highest civil/criminal court)
  - `Gerechtshof` -- Court of Appeal (with location, e.g., Gerechtshof Amsterdam)
  - `Rechtbank` -- District Court (with location, e.g., Rechtbank Den Haag)
  - `Raad van State` -- Council of State (administrative law)
  - `Centrale Raad van Beroep` -- Central Appeals Tribunal (social security/civil service)
  - `College van Beroep voor het bedrijfsleven` -- Trade and Industry Appeals Tribunal
- Date range if `date_from` and/or `date_to` are provided

### Step 3: Process and Present Results

For each result, present:

```

- **Date:** [decision date]
- **Case Number:** [zaaknummer]
- **Type:** [uitspraak / beschikking / conclusie]
- **Legal Area:** [rechtsgebied]

**Summary:**
[Concise summary of the case, key facts, and ruling]

**Key Legal Points:**
- [Relevant legal principle established or applied]
- [Relevant BW/law articles cited]

**Relevance to Query:**
[Why this case is relevant to the user's search]

---

### 2. [Next case...]
[...]
```

### Step 4: Deeper Analysis (if few results)

If fewer than 3 results are found:
- Broaden the search terms
- Try alternative Dutch legal terminology
- Search for related concepts
- Check if there are relevant Hoge Raad decisions that set precedent

### Step 5: Cross-Reference Legislation

For the most relevant cases found:
- Identify the key statutory provisions cited
- Use `legislation_search` to retrieve the current text of those provisions
- Note any legislative changes since the case was decided

### Step 6: Present Analysis

After listing results, provide:

```

- [Related legal areas to explore]
```
