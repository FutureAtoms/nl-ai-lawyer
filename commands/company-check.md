# KVK Company Registry Lookup

Look up company information in the Dutch Kamer van Koophandel (Chamber of Commerce) registry.

## Skill and Tools

Use skill: **dutch-corporate-law**

Use MCP tools:
- `kvk_search_company` -- search for companies by name or KVK number
- `kvk_get_profile` -- retrieve detailed company profile
- `kvk_get_basisprofiel` -- retrieve basic company information

**Requirement:** This command requires a valid `KVK_API_KEY` to be configured. If the API key is not available, inform the user and suggest they configure it.

## Workflow

### Step 1: Determine Search Type

- If the `company` argument is an 8-digit number, treat it as a KVK number and search directly
- If it's a name, use `kvk_search_company` to find matching companies
- Handle common variations (B.V. vs BV, with/without periods in abbreviations)

### Step 2: Search and Retrieve

Use `kvk_search_company` to find the company, then `kvk_get_profile` for detailed information.

If multiple matches are found, present all matches and ask the user to confirm which company they mean.

### Step 4: Legal Form Explanation

Provide a brief explanation of the legal form:

- **BV (Besloten Vennootschap):** Private limited company. Shareholders' liability limited to capital contribution. Governed by BW Boek 2, Title 5.
- **NV (Naamloze Vennootschap):** Public limited company. Shares may be publicly traded. Governed by BW Boek 2, Title 4.
- **Eenmanszaak:** Sole proprietorship. Owner personally liable for all debts.
- **VOF (Vennootschap onder firma):** General partnership. Partners jointly and severally liable.
- **Maatschap:** Professional partnership. Partners liable for their own share.
- **CV (Commanditaire vennootschap):** Limited partnership. Silent partners have limited liability.
- **Stichting:** Foundation. No members or shareholders. Governed by BW Boek 2, Title 6.
- **Vereniging:** Association. Governed by BW Boek 2, Title 2.
- **Cooperatie:** Cooperative. Governed by BW Boek 2, Title 3.

### Step 5: Due Diligence Flags

Check and flag any concerns:

- **Recent incorporation** (less than 1 year old) -- note for due diligence
- **Dissolved or in liquidation** -- warn the user
- **No registered address** or address at a virtual office -- flag for attention
- **Frequent name changes** -- note if visible
- **Missing SBI codes** -- may indicate incomplete registration
- **No directors listed** -- unusual, flag for investigation

### Step 6: Related Information

If relevant, note:
- Whether the company might be part of a group structure
- UBO (Ultimate Beneficial Owner) registry requirements
- Annual accounts filing obligations based on company size
- Applicable corporate governance code (if listed NV)
